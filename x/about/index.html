About qk.rs

qk.rs is a minimalist, URL-based notebook and rapid communication tool. It turns any path you type into a private, instantly editable, and shareable page. Think of it as a personal, infinitely flexible whiteboard accessible via any device.

How It Works

  • Instant Pages: Just type a path like secretpage after the main URL (or use the search feature). If the page doesn't exist, it's created instantly. If it does, you begin editing the live content.
  • Live Collaboration: All changes are saved in real time (via Firebase) and update instantly for anyone else viewing the same URL. Use it for quick collaboration, temporary shared notes, or delivering code snippets.
  • Efficiency Focus: The interface is built for speed. Use the bottom bar buttons or, even faster, dedicated keyboard shortcuts to search, copy, and navigate.
  • Long-Form Editor: The editor is a simple, distraction-free field optimized for focused writing and input.
